# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## 0.4.0 - 2026-02-22

### Added
- ✨ `service` now allows returning borrowed errors.
- ✨ Support FDs in Service impls.
- ✨ `service` methods can now return reply streams. #211
- ✨ Allow `interface` attribute on `service`.
- ✨ Implement introspection interface. #213
- ✨ Add `service` attribute macro. #76

### Breaking
- 💥 Move `Sock` from `handle` to the trait in Service. #207

### Changed
- ♻️ Split `service` tests into separate files.
- 🚚 Move `service` tests from zlink to -macros.

### Documentation
- 📝 Document `service` feature.
- 📝 Use & recommend `service` macro.
- 💡 Add missing `.` in sample code comments.

### Fixed
- 🐛 Remove `_` prefix from names for introspection.
- 🐛 Hide internal types generated by `service`.
- 🚑️ Fix a borrowing problem in service methods. #216

### Other
- 🚩 Reject FD attributes in service macro without std.
- 🚩 `service` feature now requires `introspection` feature.
- ⏪️ Revert "🚩 macros: Enable needed features on zlink dev-dep".
- 🚩 Enable needed features on zlink dev-dep.

### Changed

### Testing
